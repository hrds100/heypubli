# admin-impersonate

## What it does

Admin "Login as influencer" wrapper — shows a sticky warning banner with the impersonated name and a back-to-admin link.

## Files

- `AdminImpersonate.tsx` — warning banner + children passthrough
- `AdminImpersonate.test.tsx` — Vitest tests
- `index.ts` — public exports

## Route

Used as a wrapper layout, not a standalone route.
